# 就業規則改定案テンプレート：AI利用およびデータ保護に関する規定

> **本テンプレートの使い方：** 既存の就業規則に以下の章を追加してください。[ ] 内の項目を貴社の実情に合わせて置き換えてください。

## 第○章 AI利用およびデータ保護に関する規定

### 第○条（目的）

本章は、従業員が業務において外部の人工知能（AI）サービスを利用する際の情報セキュリティを確保し、個人情報および機密情報の漏洩を防止することを目的とする。

### 第○条（定義）

本章において、各用語の定義は以下のとおりとする。

1. **外部AIサービス：** 当社が管理しない第三者が提供するAIチャットサービス（ChatGPT、Google Gemini、Claude等を含むがこれに限らない）
2. **AIデータ保護システム：** 外部AIサービスへの入力データを自動検査し、個人情報および機密情報の漏洩を防止するソフトウェアシステム
3. **検査：** AIデータ保護システムによる入力テキストの自動的なパターン照合
4. **検出記録：** 検査の結果として記録される情報（検出カテゴリ、日時、対応アクション等。入力テキストの原文は含まない）

### 第○条（外部AIサービスの利用）

1. 従業員は、業務上必要な場合に限り、会社が許可した外部AIサービスを利用することができる。
2. 外部AIサービスの利用にあたっては、本章の規定および別途定める「AI利用ガイドライン」を遵守しなければならない。
3. 会社が許可していない外部AIサービスの業務利用は禁止する。

### 第○条（AIデータ保護システム）

1. 会社は、情報セキュリティの確保および法令遵守のため、業務で使用するPC（以下「業務端末」という）にAIデータ保護システムを導入する。
2. AIデータ保護システムは、従業員が外部AIサービスに入力するテキストを自動的に検査し、以下の情報が含まれる場合に所定の対応（マスク、警告、ブロック等）を行う。
   - 個人情報（電話番号、メールアドレス、住所、マイナンバー等）
   - 機密情報（社外秘、極秘等の機密区分が付与された情報）
   - セキュリティ上の脅威（プロンプトインジェクション等の不正な指示パターン）
3. AIデータ保護システムの検査対象は、会社が指定する外部AIサービスへの入力に限定され、それ以外のウェブ閲覧、メール、社内チャット等の通信は検査しない。
4. AIデータ保護システムは、入力テキストの原文を保存しない。検出結果（検出カテゴリ、日時、ハッシュ値等）のみを記録する。

### 第○条（検出時の対応）

1. AIデータ保護システムが個人情報を検出した場合、当該情報は自動的にマスク処理（伏字化）され、AIサービスに送信される。
2. AIデータ保護システムが機密情報またはセキュリティ上の脅威を検出した場合、当該入力は送信がブロックされ、従業員に理由が通知される。
3. ブロックが誤検知であると考える場合、従業員は第○条に定める窓口に報告し、確認を求めることができる。

### 第○条（検出記録の管理）

1. 検出記録は、[情報セキュリティ部門]が管理する。
2. 検出記録の保存期間は [90日間] とし、保存期間経過後は自動的に削除する。
3. 検出記録へのアクセスは、[情報セキュリティ部門の担当者および管理職]に限定する。
4. 検出記録は、以下の目的にのみ利用する。
   - 情報漏洩インシデントの調査
   - AIデータ保護システムの精度向上（誤検知率の改善）
   - 情報セキュリティに関する統計レポートの作成
5. 検出記録は、人事評価、懲戒処分の直接的な根拠、または勤務態度の評価には使用しない。ただし、故意による重大な情報漏洩が疑われる場合を除く。

### 第○条（従業員の権利）

1. 従業員は、自己に関する検出記録の開示を請求することができる。
2. 従業員は、検出記録に誤りがある場合、その訂正または削除を請求することができる。
3. 従業員は、誤検知（正当な入力が不当にブロックされた場合）について報告し、改善を求めることができる。
4. 前各項の請求は、[情報セキュリティ部門]に対して行うものとする。会社は、請求を受けた日から [2週間] 以内に対応する。

### 第○条（禁止事項）

従業員は、以下の行為を行ってはならない。

1. AIデータ保護システムの無効化、回避、または妨害
2. AIデータ保護システムのブラウザ拡張の削除またはアンインストール
3. AIデータ保護システムの検査を回避する目的での、暗号化、符号化、またはその他の手段による入力テキストの加工
4. 会社が許可していない外部AIサービスの利用

### 第○条（問い合わせ窓口）

本章に関する問い合わせ、誤検知の報告、開示請求等の窓口は以下のとおりとする。

- 担当部署：[情報セキュリティ部門]
- 連絡先：[メールアドレス / 内線番号]
