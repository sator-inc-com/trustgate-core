## AIデータ保護システム（TrustGate）の導入について

当社では、情報セキュリティの強化および個人情報・機密情報の漏洩防止を目的として、AIデータ保護システム「TrustGate」を [導入予定日] より導入いたします。

本通知は、個人情報の保護に関する法律第18条に基づき、取得する個人情報の利用目的等についてお知らせするものです。

---

### 1. 導入の目的

業務で使用する外部AIサービス（ChatGPT、Google Gemini、Claude等）への入力データを検査し、以下のリスクから当社および従業員の皆様を保護することを目的とします。

- 顧客の個人情報（氏名、電話番号、メールアドレス等）の意図しない外部送信
- 機密情報（社外秘資料等）の外部AIサービスへの流出
- プロンプトインジェクション等のセキュリティ攻撃

### 2. 検査の対象

| 項目 | 内容 |
|------|------|
| 対象サービス | [ChatGPT (chatgpt.com)、Google Gemini (gemini.google.com)、Claude (claude.ai)] |
| 対象データ | 上記サービスへの入力テキストおよびファイルアップロード |
| 対象外 | 上記以外のウェブサイト閲覧、メール、社内チャット等 |
| 対象者 | [全従業員 / 対象部門名] |

**本システムは、指定されたAIサービスへの入力のみを検査します。それ以外のウェブ閲覧やメール等の活動は一切検査しません。**

### 3. 検査の仕組み

```
ブラウザ拡張 → Desktop Agent（お使いのPC内） → 検査結果のみ記録
                      ↑
            テキストの検査はPC内で完了
            入力テキストの原文は保存されません
```

- ブラウザ拡張（Chrome拡張機能）が対象AIサービスへの入力を検知します
- 入力テキストはお使いのPC上の Desktop Agent に送信され、検査されます
- **入力テキストの原文は一切保存されません。** 検出結果（「電話番号が検出された」等のカテゴリ情報）とハッシュ値（原文を復元できない暗号化値）のみが記録されます
- 検査はお使いのPC内で完結します。外部サーバーへ原文が送信されることはありません

### 4. 検出時の動作

検査の結果、以下の対応が自動的に行われます：

| 検出内容 | 動作 | 例 |
|---------|------|---|
| 個人情報（電話番号、メール等） | 自動マスク（伏字化） | `090-****-5678` |
| 機密キーワード（社外秘等） | 送信ブロック | 入力が送信されず、通知が表示されます |
| 不正な指示パターン | 送信ブロック | 入力が送信されず、通知が表示されます |

ブロックされた場合、画面上に理由が表示されます。誤検知の場合は、後述の問い合わせ窓口にご連絡ください。

### 5. 記録される情報

| 記録項目 | 記録の有無 | 備考 |
|---------|----------|------|
| 入力テキストの原文 | **記録しません** | — |
| 検出カテゴリ（例：「電話番号」） | 記録します | 何が検出されたかの分類のみ |
| 検出日時 | 記録します | — |
| ユーザーID | 記録します | 社内ユーザーID |
| 対応アクション（マスク/ブロック等） | 記録します | — |
| 入力テキストのハッシュ値 | 記録します | 原文を復元することはできません |

### 6. 記録の保存期間

検出記録は [90日間 / 180日間 / 1年間] 保存した後、自動的に削除されます。

### 7. 皆様の権利

従業員の皆様には、以下の権利があります：

- **開示請求権：** ご自身に関する検出記録の開示を請求できます
- **訂正・削除請求権：** 誤った記録の訂正または削除を請求できます
- **フィードバック権：** 誤検知（正当な入力がブロックされた場合）を報告し、改善を求めることができます
