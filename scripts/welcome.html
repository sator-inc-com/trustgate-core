TrustGate — AI Zero Trust Gateway

Inspect and control AI input/output in real time.

This installer will set up the following components:

  • TrustGate Agent — Text inspection, policy enforcement, LLM proxy
  • Tray Manager — System tray status indicator (Apple Silicon only)
  • Default configuration — Standalone mode with sample policies

The Agent will be registered as a system service and start automatically on boot. It listens on localhost:8787.

After installation, verify with:

curl -s http://localhost:8787/v1/health | jq .
